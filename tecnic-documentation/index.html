HTML Semântico

HTML semântico utiliza elementos que possuem significado claro.

Isso melhora a acessibilidade e o SEO.

<header> <main> <section>
  • Melhora a leitura do código
  • Ajuda leitores de tela
  • Facilita manutenção
CSS Responsivo

CSS responsivo adapta o layout a diferentes tamanhos de tela.

É essencial para celulares, tablets e desktops.

display: flex;
  • Flexbox
  • Grid Layout
Media Queries

Media queries permitem aplicar estilos conforme o tamanho da tela.

São fundamentais no design responsivo.

@media (max-width: 800px)
  • Breakpoints
  • Layouts adaptáveis
